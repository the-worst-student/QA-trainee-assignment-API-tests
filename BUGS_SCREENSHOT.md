# Баги на скриншоте

- P1 — в выдаче отображаются объявления с ценой вне выбранного диапазона `1000–30000 ₽` (`40 000 ₽` и `500 000 ₽`).
- P1 — в выдаче `Котята в Москве` отображается объявление из другого города (`Новосибирск`).
- P1 — при выбранной породе `Персидская` в выдаче отображается объявление другой породы (`Экзотический короткошерстный котёнок`).
- P1 — при выбранном поле `Мальчик` в выдаче отображается объявление `Персидский котёнок девочка`.
- P1 — на странице отображается сообщение `Ошибка, обновите страницу`, хотя выдача и карточки загружены.
- P2 — при наведении на иконку сердца отображается подсказка `Не показывайте мне похожие объявления`, которая не соответствует ожидаемому действию элемента `Избранное`.
- P2 — в интерфейсе одновременно отображаются противоречивые географические признаки: заголовок `Котята в Москве`, при этом рядом доступен переключатель `Сначала из Можайска`.
- P2 — в интерфейсе одновременно отображаются противоречивые количества результатов: в заголовке указано `19`, а кнопка применения фильтров показывает `Показать 8 объявлений`.
- P2 — всплывающая подсказка перекрывает контент карточек и мешает просмотру информации.
- P3 — в фильтре возраста для котят используется некорректное значение `Щенок`.
